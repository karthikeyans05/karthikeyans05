## 🚀 Featured Projects

### SafeDrive Vision — Android Driver Drowsiness Detection
> **Production-ready · Fully Offline**

A real-time driver safety app that monitors eye closure and head tilt using Google ML Kit Face Detection + CameraX. Fires sound and vibration alerts before drowsiness becomes dangerous — all on-device with zero network calls.

**Stack:** Kotlin · Android · CameraX · ML Kit · TensorFlow Lite · MVVM · Kotlin Coroutines  
**Highlights:** 100% offline inference · Eye closure detection at 2s threshold · Head tilt at 3s threshold · Custom Canvas bounding box overlay · Auto exposure compensation · Clean MVVM architecture

---

### ExpenseFlow — Full Stack Expense Tracker
> **Deployed · Docker + Render**

Full-stack personal finance platform with category donut charts, monthly trend bar charts, budget alerts, and full-text expense search. Complete REST API with CRUD operations and input validation front-to-back.

**Stack:** Java · Spring Boot · JavaScript · HTML/CSS · MySQL · Docker · Render  
**Highlights:** JPA/Hibernate ORM · Maven build · docker-compose one-command setup · Responsive mobile UI · Analytics page with payment method breakdowns

---

### AI Chatbot — ChatGPT-Style Web App
> **Full Stack · LLM Integration**

A conversational AI web application with a ChatGPT-inspired UI. Node.js + Express backend integrates OpenRouter LLM API with MongoDB-persisted conversation history across sessions.

**Stack:** Node.js · Express · MongoDB · JavaScript · HTML/CSS · OpenRouter API  
**Highlights:** Full-screen dark UI · Keyboard shortcuts (Enter to send) · Persistent conversation history · Real-time streaming responses

---

### Attend IQ — Smart Attendance Management System
> **Web App · Educational Tech**

Attendance management platform for educational institutions. Intelligent tracking with insights and reporting that reduces manual overhead for faculty.

**Stack:** JavaScript · Node.js · MySQL
